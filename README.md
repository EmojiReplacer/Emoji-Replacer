Replace your boring old stock emojis fast & easy 🥳🙌💚

- Clean 🆕 Material you UI 📱
- Dynamic colors (Android 12+)  🎨
- Create a Magisk module 🤐
- Light and Dark mode 🌝🌚
- The latest Emojis 😀
- Choose from a variety of Emojis 😁
- Select your own emoji fonts from internal storage 😅
